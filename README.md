## 功能

| 功能 | 说明 |
|------|------|
| 工作模式 | 25 分钟专注计时 |
| 短休息 | 5 分钟短暂放松 |
| 长休息 | 15 分钟深度休息，每完成 4 个番茄自动触发 |
| 进度环 | SVG 圆环动画，实时显示剩余时间比例 |
| 声音提醒 | Web Audio API 合成，到点自动播放 |
| 桌面通知 | 浏览器原生通知，切换窗口也能收到提醒 |
| 键盘快捷键 | `Space` 开始/暂停 · `R` 重置 · `1/2/3` 切换模式 |

## 使用方式

双击 `pomodoro.html` 在浏览器中打开即可使用，无需安装任何依赖。

```
pomodoro.html    # 双击打开
```

## 快捷键

| 按键 | 功能 |
|------|------|
| `空格` | 开始 / 暂停计时 |
| `R` | 重置为当前模式完整时长 |
| `1` | 切换到工作模式（25 min） |
| `2` | 切换到短休息（5 min） |
| `3` | 切换到长休息（15 min） |

## 技术实现

- 纯 HTML/CSS/JS 单文件，零依赖
- CSS 变量主题系统，修改变量即可换肤
- SVG `stroke-dasharray` / `stroke-dashoffset` 实现圆形进度动画
- Web Audio API `OscillatorNode` + `GainNode` 合成提示音
- 状态机模式管理计时器生命周期（idle → running → paused → finished）
